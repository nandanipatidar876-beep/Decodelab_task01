# Decodelab_task01
📝 To-Do List Application

Project Overview

This project is a simple command-line To-Do List application developed using Python. It allows users to add tasks, view their tasks, and exit the application through an interactive menu-driven interface.

The project demonstrates the use of fundamental Python concepts such as lists, loops, conditional statements, and user input handling.

---

Objectives

- Create a user-friendly task management program.
- Store multiple tasks using Python lists.
- Display tasks in an organized manner.
- Practice basic programming logic and control structures.

---

Features

✅ Add new tasks to the list

✅ View all saved tasks

✅ Menu-driven interface

✅ Input validation for incorrect choices

✅ Simple and easy-to-use design

---

Technologies Used

- Python 3
- Command Line Interface (CLI)

---

Concepts Implemented

- Lists
- While Loops
- Conditional Statements (if-elif-else)
- User Input
- List Traversal using Loops

---

Program Flow

1. Display the main menu.
2. User selects an option.
3. If the user chooses:
   - 1 → Add a new task.
   - 2 → View all existing tasks.
   - 3 → Exit the application.
4. Repeat until the user exits.
